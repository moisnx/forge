GUIDE

Getting Started

Learn how to build your first static site with Forge in just a few minutes. No configuration needed.

1

Installation

Clone the Forge repository and build from source. Forge is written in C++ and compiles to a single binary.

terminal 
git clone https://github.com/yourusername/forge.git
cd forge
make

# Optional: add to PATH
sudo cp forge /usr/local/bin/

Requirements

You'll need a C++17 compatible compiler and make installed on your system.

2

Create Your First Site

Initialize a new Forge project. This creates the basic folder structure and example files to get you started.

terminal 
forge init my-awesome-site
cd my-awesome-site

Start the Dev Server

Launch the development server with hot-reload. Forge watches your files and automatically rebuilds when you make changes.

terminal 
forge serve

Open your browser to http://localhost:8080 and you'll see your site. Make changes to any file and watch them appear instantly!

4

Create Content

Add a new page by creating a markdown file in the content/ directory with YAML frontmatter.

content/my-first-post.md 
---
title: My First Post
date: 2025-01-15
author: Your Name
---

# Welcome to Forge

This is my first page built with **Forge**!

It's fast, simple, and powerful.

Pro Tip

5

Build for Production

When you're ready to deploy, build the production version. This generates optimized static files in the build/ directory.
